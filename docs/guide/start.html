# 开始

本框架需要你本地安装 Node.js (opens new window) ，同时我们强烈建议你安装并使用 Yarn (opens new window) 做为包管理工具。

做好准备工作后，依次执行下面的命令：

# 进入项目根目录
cd fantastic-admin
# 安装项目依赖
yarn
# 运行项目
yarn run serve

运行成功后，会自动访问页面，默认情况下，地址为 http://localhost:8080
