# 基础组件

基础组件由 ElementUI 提供，其中包含按钮、文字链接、单选框、多选框、输入框、计数器、级联选择器、开关、滑块等 50+ 个组件。

阅读并学习请查看 ElementUI 官方文档 (opens new window)。
