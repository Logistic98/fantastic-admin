# 关于 Fantastic-admin

## 这是什么

一款开箱即用的 Vue 中后台管理系统模版，基于 [vue-automation](https://hooray.github.io/vue-automation) & [Element](https://element.eleme.cn)

## 特点

- 丰富的布局模式，覆盖市面上各种中后台布局
- 提供主题配置文件，轻松实现个性化定制
- 根据路由配置自动生成导航栏
- 支持全方位权限验证
